# University Data Analysis (LINQ)

This project is a C# console application designed to practice LINQ queries on a university data model.

## Prerequisites

* [.NET SDK](https://dotnet.microsoft.com/download) (Version 6.0 or higher)
* JetBrains Rider (or any preferred C# IDE/Text Editor)

## How to Run

### Using the Terminal (Recommended for Rider)

1. Open the **Terminal** in Rider (usually at the bottom or press `Alt + F12`).
2. Navigate to the project folder (the one containing the `.csproj` file):
   ```powershell
   cd LinqConsoleLab.EN
3.Run the application using the .NET CLI
   ```powershell
dotnet run
